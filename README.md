## Sobre o Projeto

Sistema web completo para empresas de locação de motos, cobrindo desde o cadastro do cliente até o controle financeiro.

## Funcionalidades

### Painel Administrativo
- Gestão completa de motos (cadastro, status, manutenção)
- Contratos com assinatura digital via **Clicksign**
- Controle de pagamentos e fluxo de caixa
- Gerenciamento de convites para novos clientes
- Bot de atendimento via **WhatsApp** (Evolution API)

### Painel do Cliente
- Acompanhamento do contrato ativo
- Histórico de pagamentos
- Controle de KM e troca de óleo
- Progresso de propriedade da moto

### Integrações
| Serviço | Função |
|---|---|
| **Mercado Pago** | Checkout de pagamento inicial |
| **Clicksign** | Assinatura digital de contratos |
| **Evolution API** | WhatsApp bot e notificações |
| **Resend** | E-mails transacionais |

### Outros
- Landing page comercial com planos
- Cadastro por convite com verificação WhatsApp + e-mail
- Validação de CPF em tempo real
- Responsivo para mobile

## Stack Tecnológica

| Camada | Tecnologia |
|---|---|
| Frontend | React 18, TypeScript, Vite, Tailwind CSS, shadcn/ui |
| Estado | React Query, React Router |
| Backend | Supabase (Auth, Database, Storage, Edge Functions) |
| Pagamentos | Mercado Pago API |
| Contratos | Clicksign API |
| Comunicação | Evolution API (WhatsApp), Resend (E-mail) |
